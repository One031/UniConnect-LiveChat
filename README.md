# UniConnect-LiveChat

A real-time chat web application built with Jakarta EE, JSP/Servlets, and MySQL.

## Features

* User registration and login with secure password hashing
* Real-time messaging between users
* MySQL database integration
* Built with Jakarta EE and deployed on GlassFish

## Tech Stack

* Jakarta EE, Java Servlets, JSP
* MySQL
* GlassFish Server
* Maven
